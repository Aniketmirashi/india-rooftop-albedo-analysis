# Data

## Google Rooftop Albedo Dataset

The primary dataset used in this study is Google's **Open Buildings Rooftop Albedo Dataset**, which provides rooftop-level albedo estimates for urban areas worldwide.

Due to GitHub's file size limitations, the original dataset is **not included** in this repository.

## Download

Please download the India dataset directly from Google's Open Buildings data portal:

 India specific data: https://storage.googleapis.com/eie-cool-roofs-public/india.zip

---

## Dataset Information

| Attribute | Description |
|-----------|-------------|
| Dataset | Google  Heat Resilience data |
| Region | India |
| Format | zip/CSV |
| Records | ~3.34 Million Rooftop Observations |
| Variables | `mean_albedo`, `lat`, `lng` |
